Aprendendo Flexbox

O que é Flexbox?

Flexbox é um modelo de layout do CSS usado para alinhar e distribuir espaços entre os itens em um container

Como usar?

Basta aplicar display: flex ao elemento pai e os filhos, automaticamente, se organizam em linha ou coluna

Por que usar?

Facilita o desenvolvimento de layout responsivos e linhamento de elementos na tela.
